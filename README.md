# 图片处理工具 / Image Processing Tools

一个功能强大的在线图片处理工具，支持图片放大和添加文字功能。

A powerful online image processing tool with image enlargement and text addition features.

## 功能特性 / Features

### 图片放大 / Image Enlarge
- 支持2x、3x、4x、5x放大倍数
- 高质量图像平滑算法
- 支持JPG、PNG、GIF格式
- 拖拽上传支持

### 添加文字 / Add Text
- 自定义文字内容
- 可调整字体大小（12-72px）
- 颜色选择器
- 多种字体选择
- 点击选择文字位置
- 文字阴影效果

## 多语言支持 / Multi-language Support

- **中文版**: `index.html` (根目录)
- **English**: `en/index.html` (en目录)

## 使用方法 / Usage

1. 直接打开 `index.html` 或 `en/index.html` 在浏览器中使用
2. 支持拖拽上传图片文件
3. 根据提示操作完成图片处理
4. 处理完成后可直接下载结果

## 技术特点 / Technical Features

- 纯前端实现，无需服务器
- 响应式设计，支持PC、手机、平板
- 现代CSS和JavaScript
- Canvas API图像处理
- 高质量图像输出

## 浏览器兼容性 / Browser Compatibility

- Chrome (推荐)
- Firefox
- Safari
- Edge
- 移动端浏览器
